## 分项目clone compose

```sh
# 确认当前目录是/home/ecs-user
pwd

# 根据情况建立子目录。
mkdir demo

git clone git@github.com:le2-inf/docker-compose.git
git clone git@github.com:cd-le2/docker-compose.git

cd docker-compose
```

## 设置环境变量

```shell
# 查看存在的环境变量
ls -l env/

echo "env_mode=pro" > .env
#vim .env

# 
make env
```

## 设置docker网络

```shell
make setup_docker_network
```

## 运行
```sh
# 分环境运行
make pro opt=_
make pro stg opt=_
make pro stg dev opt=_
make pro stg dev local opt=_
```
